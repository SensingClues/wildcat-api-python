WildcatApi
==========
Info
----

| This document contains all the functions that can be called by the api.
- Blue box is the class
- grey box are the function you can call on the class

| The api is a class that has to be initalized. Initializing this class is done with the user_name and password.
  Afterwards, you can use this class to call the different functions. Moreover,

.. code-block:: python

   api = WildcatApi(username,password) # initialize
   api.get_groups() # get group data
